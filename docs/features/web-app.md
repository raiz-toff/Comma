# The Web App

Comma runs as an installable Progressive Web App at [comma-psi.vercel.app](https://comma-psi.vercel.app) — a full companion to the phone app for logging and reviewing your work from any browser.

---

## Installing

Open the app in a modern browser and install it so it runs in its own window and works offline:

| Platform | How |
|---|---|
| Chrome / Edge (desktop) | The install icon in the address bar, or menu → *Install COMMA*. |
| Chrome (Android) | Menu → *Install app*. |
| Safari (iOS) | Share → *Add to Home Screen*. |

You can also start the install from **Settings → About → Install COMMA**.

---

## What works the same

Almost everything the phone app does, the web app does too:

- The dashboard
- Shifts
- Expenses
- The tax centre
- Goals
- Analytics
- Reports
- Vehicles
- Google Drive backup **and** cloud sync

---

## What's different from the phone

Three things behave differently in a browser:

| Area | On the web |
|---|---|
| GPS tracking | Runs only while the tab is open — there's no background service. Use the phone for real driving. |
| Notifications | In-page reminders rather than true background push. |
| Receipts | File-picker only; there's no native camera. |

Because there is no background service, closing the tab stops distance tracking. The web app is the right tool for review and desk-based logging, not for recording a shift while you drive.

---

## Two vaults, kept in sync

The phone app and the web app are **separate local vaults** — each keeps its own copy of your data. Connecting **both** to the **same Google Drive** keeps them in step: log a shift on the phone, open the browser, and it's there.

This is real, shipped sync — not a workaround and not "no sync". The full mechanics are in [Backup & Sync](../backup-and-sync/overview.md).

---

## Web-only conveniences

The browser adds a handful of things the phone doesn't:

- A **CSV import wizard** for bringing in historical data.
- Entering a **whole week of shifts** in one form.
- A **print view** for producing a PDF.
- **Keyboard shortcuts** for power users.
- **Fullscreen / focus mode** to cut distractions while you review.
- **Offline operation** — it installs and works with no network, and exports made offline are queued and sent when you reconnect.
