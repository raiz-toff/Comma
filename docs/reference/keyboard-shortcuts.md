# Keyboard shortcuts

The keyboard shortcuts available in Comma, primarily for the web app on a desktop keyboard.

---

## Shortcuts

Navigation shortcuts beginning with `g` are chords: press `g`, release, then press the second key.

| Shortcut | Action |
|---|---|
| `g` then `d` | Go to the dashboard. |
| `g` then `s` | Go to shifts. |
| `g` then `t` | Go to the tax workspace. |
| `g` then `v` | Go to vehicles. |
| `Cmd/Ctrl` + `K` or `/` | Open search. |
| `Esc` | Close the open overlay, dialog, or search. |

---

## Where to find them

On the web app the full list is in **Settings, About, Keyboard shortcuts**. On the phone app the same reference appears within Settings; shortcuts apply when the settings viewer is used with an attached keyboard. See [Settings](settings.md).
